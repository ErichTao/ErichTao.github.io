GitHub 排行精选

1、Motrix

这是一款全能的下载工具，支持下载 HTTP、FTP、BT、磁力链、百度网盘等资源。界面简洁易用，漂亮大方！

相关文章：强烈推荐：高颜值、不限速下载工具

详情请点击：https://github.com/agalwood/Motrix

2、996.ICU

截止目前 996.ICU 已经 110k star 了。

相关文章：国内程序员这次牛逼了，「微博热搜」，「GitHub 」双双霸榜了

详情请点击：https://github.com/996icu/996.ICU

3、Android-ZBLibrary

这是一个 MVP 架构的 Android 快速开发框架，该框架提供了一套开发标准（View，Data，Event）以及模板和工具类并规范代码。封装层级少，简单高效兼容性好。

相关文章：Demo最全面、注释最详细、使用最简单、代码最严谨的 Android 开源 UI 框架

详情请点击：https://github.com/TommyLemon/Android-ZBLibrary

4、the-craft-of-selfteaching

作者自称《自学是门手艺》可能是当前世界上最硬核的鸡汤书。从我对该书部分阅读来看，书本更多的是传达一些理念，应该如何看待自学这个事情，因为学习这件事会相伴我们一生，尤其对于我们工程师来说，技术更迭如此之快，不学习怎么跟得上呢。

相关文章：中国区块链第一人教你如何自学

详情请点击：https://github.com/selfteaching/the-craft-of-selfteaching

5、USTC-Course

这个开源项目收集了中国科学技术大学 的关于计算机方面的课程资源，笔记，经验总结等各种资料。

通过了解这个开源项目收集的课程资源，你就可以了解中国顶级大学关于计算机课程相关的内容，知道好大学的都是如何进行计算机相关课程学习的。

相关文章：这是一个牛逼大学的课程资源，程序员的你是来自这个大学么？

详情请点击：https://github.com/USTC-Resource/USTC-Course

高颜值、不限速下载工具

Motrix，它是一款全能的下载工具，支持下载 HTTP、FTP、BT、磁力链、百度网盘等资源。基于 Electron + Vue，支持 Windows/Mac/Linux

开源项目地址：[https://github.com/agalwood/Motrix

开源项目作者：Dr_rOot

Android 开源 UI 框架

Android-ZBLibrary，MVP 架构的 Android 快速开发框架，该框架提供了一套开发标准（View，Data，Event）以及模板和工具类并规范代码。封装层级少，简单高效兼容性好。

Android-ZBLibrary 也可使用 APIJSONAuto 自动化接口管理工具来生成接口相关代码：

  • 自动生成封装请求 JSON 的代码

  • 自动生成解析结果 JSON 的代码

  • 自动生成 Modle/JavaBean

开源项目地址：TommyLemon/Android-ZBLibrary

开源项目作者：TommyLemon
